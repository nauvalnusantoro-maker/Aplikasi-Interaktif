Distribusi Token
Regular Expression (Regex) adalah pola teks yang digunakan untuk mencocokkan string. Lexer JavaScript menggunakan urutan regex berikut (prioritas dari atas ke bawah). Token pertama yang cocok akan digunakan.
NFA (Nondeterministic Finite Automaton) adalah model teoritis dari regex. Setiap pola regex dapat dikonversi ke NFA menggunakan konstruksi Thompson. NFA memiliki ε-transitions (transisi kosong) dan beberapa state aktif sekaligus.
Skema NFA untuk Token JavaScript
DFA (Deterministic Finite Automaton) diperoleh dari NFA melalui proses subset construction. Setiap state DFA merepresentasikan satu set state NFA. DFA hanya memiliki tepat satu transisi per simbol input dari setiap state — inilah yang membuat lexer efisien.

Tabel Transisi DFA merepresentasikan fungsi transisi δ(q, a) → q' secara eksplisit. Baris = state saat ini, Kolom = simbol input/kategori. * menandai accept state. State q0 adalah state awal. Nilai — berarti dead/trap state.
